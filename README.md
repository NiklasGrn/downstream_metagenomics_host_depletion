
# Metagenomics Downstream Analysis  

A collection of R files used for downstream analysis of metagenomics data.




## Usage

All R files are self contained and the data used to generate the corresponding plots is also present in this GitHub.

The sub folders grapes_... were used in the analysis of the grape samples while the subfolder zymo_standard was used in the analysis of the Zymo HMW DNA Standard sample. 

## Used By

These R files were used while writing the thesis "Comparison of Adaptive Sampling and Biochemical Host Depletion Strategies for Whole Genome Shotgun Sequencing and 16S Nanopore Metagenomics Workflows" by Niklas Grundner
